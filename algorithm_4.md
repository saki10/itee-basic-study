# モジュールの独立性
+ モジュール
+ モジュール結合度
　

+ がっつり結合（独立性が低い）は　悪いモジュ
ール
  + 内部結合　他のモジュール内を参照する
  + 共通結合　外部宣伝されたデータ構造（データのまとまり）を参照する
  + 外部結合　外部宣言されたデータを参照する
+ うっすら結合（独立性が高い）は　良いモジュール
  + スタンプ結合 データ構造（データのまとまり）を引数として渡す
  + 制御結合　モジュールの実行順序を制御するデータ（制御パラメータ）を引数として渡す
